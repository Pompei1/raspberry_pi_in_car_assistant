# 🚗 Assistente Veicular com Raspberry Pi

Este projeto tem como objetivo criar um assistente veicular pessoal utilizando
Raspberry Pi, com foco em comandos de voz, rotas, agenda e dados do veículo.

## 🎯 Objetivo
Criar um assistente veicular funcional, priorizando:
- Uso offline sempre que possível
- Baixa distração ao motorista
- Arquitetura modular e escalável

## 📌 Status do Projeto
🚧 Em desenvolvimento (fase inicial de estudos)

## 🛠 Tecnologias planejadas
- Python
- Raspberry Pi OS
- HTML, CSS e JavaScript
- Bluetooth (OBD2)
- Speech-to-Text offline
